Para instalar a ferramenta htmlcxx:
  cd htmlcxx-0.86
  sudo ./configure && sudo make && sudo make install

Para compilar o indexador, com a ferramenta htmlcxx instalada:
  make indexer

Para executar o indexador:
  ./exe

Para compilar o processador de consultas, com a ferramenta htmlcxx instalada:
  make qProc

Para executar o processador de consultas:
  ./queries
